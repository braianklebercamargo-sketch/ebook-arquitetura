# O Guia Prático do Arquiteto - Landing Page

Uma landing page de alta conversão desenvolvida para apresentar o material educativo "O Guia Prático do Arquiteto". O projeto visa conectar a teoria da faculdade com a prática do mercado real para estudantes e recém-formados em arquitetura.

## 🚀 Funcionalidades

- **Design Premium**: Interface focada em uma estética moderna, elegante e profissional com esquema de cores *Cosmic Slate* e escala tonal em dourado.
- **Captura de Leads**: Modal integrado no CTA principal que capta Nome, Email e WhatsApp antes de direcionar o usuário com fluido natural para o gateway de pagamento.
- **Animações Fluidas**: Microinterações e scroll animations desenvolvidos com `motion/react` para prender a atenção e guiar o foco dos usuários.
- **Carrosséis Interativos**: Componente horizontal arrastável (drag-to-scroll e touch) dinâmico, utilizado para imagens de obras/projetos reais e também para depoimentos (feedbacks) de alunos.
- **Gatilhos Mentais**: Estrutura de copy e renderização englobando provas sociais robustas, cronômetros de escassez e comparativo visual dinâmico (Onde você está vs Onde você vai chegar).
- **Totalmente Responsivo**: Otimizado para conversão máxima adaptando a tela para resoluções *mobile*, *tablet* e *desktop*.

## 🛠️ Tecnologias Utilizadas

- **[React 18+](https://react.dev/)**: Biblioteca para a criação da UI.
- **[TypeScript](https://www.typescriptlang.org/)**: Tipagem robusta para prevenção de erros.
- **[Vite](https://vitejs.dev/)**: Ferramenta de desenvolvimento ultrarrápida.
- **[Tailwind CSS](https://tailwindcss.com/)**: Motor de estilização responsivo e baseado em utilitários.
- **[Motion](https://motion.dev/)**: O motor de animação base por trás da progressão do layout.
- **[Lucide React](https://lucide.dev/)**: Coleção consistente de ícones vetoriais.

## 📦 Como rodar o projeto localmente

1. Clone este repositório:
```bash
git clone <url-do-repositorio>
```

2. Entre no diretório da aplicação:
```bash
cd projeto-arquiteto
```

3. Instale as dependências com NPM:
```bash
npm install
```

4. Inicie o ambiente de desenvolvimento local:
```bash
npm run dev
```

A aplicação será iniciada localmente (geralmente em `http://localhost:3000` ou outra porta especificada no servidor do Vite).
